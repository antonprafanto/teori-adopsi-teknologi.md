## 🎯 Tentang Repositori Ini

Repositori ini berisi materi lengkap tentang **Teori Adopsi Teknologi** (*Technology Adoption Theory*) yang disusun secara sistematis dan mudah dipahami. Materi ini dirancang untuk:

- 🎓 **Mahasiswa** S1/S2 yang menulis skripsi/tesis berbasis adopsi teknologi
- 🔬 **Peneliti** yang ingin memahami dan menerapkan teori adopsi
- 💼 **Praktisi** yang ingin memahami mengapa pengguna menerima atau menolak teknologi
- 📋 **Pembuat Kebijakan** yang ingin mendorong adopsi teknologi di masyarakat

## 🚀 Cara Menggunakan Materi Ini

Mulai dari urutan bab secara berurutan:
```
BAB-01 → BAB-02 → BAB-03 → BAB-04 → BAB-05 → BAB-06
```

### Untuk Mahasiswa yang Sedang Skripsi/Tesis
Fokus pada jalur ini:
```
BAB-01 → BAB-06 (TAM) atau BAB-07 (UTAUT) → BAB-13 (Perbandingan) 
→ BAB-28 (Metodologi) → BAB-29 (Instrumen) → BAB-30 (SEM/PLS) → BAB-32 (Template)
```

### Untuk Praktisi
Fokus pada jalur ini:
```
BAB-01 → BAB-15 (Faktor) → BAB-16 (Hambatan) → BAB-25 (Per Sektor) → BAB-27 (Change Management)
```

```
BAB-24 (Konteks Indonesia) → BAB-25 (Per Sektor) → BAB-19 (Digital Divide)
```
