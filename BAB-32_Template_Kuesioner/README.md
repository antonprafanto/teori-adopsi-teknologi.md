# BAB-32: Template Kuesioner Penelitian Adopsi Teknologi

> *"Template ini bukan untuk disalin mentah-mentah — melainkan sebagai titik awal yang telah tervalidasi untuk Anda adaptasi sesuai konteks penelitian Anda."*

---

## 🎯 Tujuan Pembelajaran

Setelah membaca bab ini, pembaca diharapkan mampu:
- Menggunakan template item kuesioner dari berbagai teori adopsi
- Mengadaptasi item kuesioner sesuai dengan teknologi dan populasi yang diteliti
- Merancang kuesioner yang lengkap dan siap digunakan
- Menghindari kesalahan umum dalam penggunaan item tervalidasi

---

## ⚠️ Panduan Penggunaan

Sebelum menggunakan template di bawah ini, perhatikan hal-hal berikut:

> [!IMPORTANT]
> 1. **Selalu cite sumber asli** dari setiap instrumen yang Anda gunakan — bukan bab ini
> 2. **Ganti [NAMA SISTEM/APLIKASI]** dengan nama teknologi yang Anda teliti
> 3. **Lakukan pilot study** (n=30) sebelum pengumpulan data utama
> 4. **Pastikan terjemahan** mempertahankan makna aslinya (lihat [BAB-29](../BAB-29_Instrumen_dan_Skala_Pengukuran/README.md))
> 5. **Adaptasi jika perlu** — tidak semua item cocok untuk semua konteks

## 32.1 Template TAM (Davis, 1989)

**Skala:** 1 (Sangat Tidak Setuju) — 5 (Sangat Setuju)  
**Sumber:** Davis, F. D. (1989). *MIS Quarterly*, 13(3), 319–340.

### Perceived Usefulness (PU)

| No | Item | Kode |
|---|---|---|
| 1 | Menggunakan [NAMA SISTEM] meningkatkan kinerja saya dalam pekerjaan | PU1 |
| 2 | Menggunakan [NAMA SISTEM] meningkatkan produktivitas saya | PU2 |
| 3 | Menggunakan [NAMA SISTEM] meningkatkan efektivitas saya | PU3 |
| 4 | Menggunakan [NAMA SISTEM] membuat pekerjaan saya lebih mudah | PU4 |
| 5 | [NAMA SISTEM] berguna untuk pekerjaan saya | PU5 |

### Perceived Ease of Use (PEOU)

| No | Item | Kode |
|---|---|---|
| 6 | Belajar mengoperasikan [NAMA SISTEM] mudah bagi saya | PEOU1 |
| 7 | Saya mudah membuat [NAMA SISTEM] melakukan apa yang saya inginkan | PEOU2 |
| 8 | Interaksi saya dengan [NAMA SISTEM] jelas dan mudah dipahami | PEOU3 |
| 9 | Saya merasa [NAMA SISTEM] fleksibel untuk digunakan | PEOU4 |
| 10 | Mudah bagi saya untuk menjadi terampil menggunakan [NAMA SISTEM] | PEOU5 |
| 11 | Saya merasa [NAMA SISTEM] mudah digunakan | PEOU6 |

### Behavioral Intention (BI) — TAM

| No | Item | Kode |
|---|---|---|
| 12 | Saya berniat menggunakan [NAMA SISTEM] di masa mendatang | BI1 |
| 13 | Saya memprediksi akan menggunakan [NAMA SISTEM] di masa mendatang | BI2 |
| 14 | Saya berencana menggunakan [NAMA SISTEM] secara rutin | BI3 |

## 32.2 Template UTAUT (Venkatesh et al., 2003)

**Sumber:** Venkatesh, V., Morris, M. G., Davis, G. B., & Davis, F. D. (2003). *MIS Quarterly*, 27(3), 425–478.

### Performance Expectancy (PE)

| No | Item | Kode |
|---|---|---|
| 1 | Saya merasa [NAMA SISTEM] berguna dalam pekerjaan saya | PE1 |
| 2 | Menggunakan [NAMA SISTEM] membantu saya menyelesaikan pekerjaan lebih cepat | PE2 |
| 3 | Menggunakan [NAMA SISTEM] meningkatkan produktivitas saya | PE3 |
| 4 | Jika saya menggunakan [NAMA SISTEM], kemungkinan saya mendapat kenaikan gaji/promosi lebih besar | PE4 |

### Effort Expectancy (EE)

| No | Item | Kode |
|---|---|---|
| 5 | Interaksi saya dengan [NAMA SISTEM] jelas dan mudah dipahami | EE1 |
| 6 | Mudah bagi saya untuk menjadi terampil menggunakan [NAMA SISTEM] | EE2 |
| 7 | Saya merasa [NAMA SISTEM] mudah digunakan | EE3 |
| 8 | Belajar mengoperasikan [NAMA SISTEM] mudah bagi saya | EE4 |

### Social Influence (SI)

| No | Item | Kode |
|---|---|---|
| 9 | Orang-orang yang penting bagi saya berpikir bahwa saya harus menggunakan [NAMA SISTEM] | SI1 |
| 10 | Orang-orang yang berpengaruh dalam kehidupan saya berpendapat bahwa saya harus menggunakan [NAMA SISTEM] | SI2 |
| 11 | Orang-orang di lingkungan saya telah membantu saya dalam menggunakan [NAMA SISTEM] | SI3 |

### Facilitating Conditions (FC)

| No | Item | Kode |
|---|---|---|
| 12 | Saya memiliki sumber daya yang diperlukan untuk menggunakan [NAMA SISTEM] | FC1 |
| 13 | Saya memiliki pengetahuan yang diperlukan untuk menggunakan [NAMA SISTEM] | FC2 |
| 14 | [NAMA SISTEM] tidak kompatibel dengan sistem lain yang saya gunakan *(R)* | FC3 |
| 15 | Ada orang/tim yang tersedia untuk membantu saya jika mengalami kesulitan | FC4 |

*(R) = item reverse-scored*

## 32.4 Template Trust (McKnight et al., 2002)

**Sumber:** McKnight, D. H., Choudhury, V., & Kacmar, C. (2002). *Information Systems Research*, 13(3), 334–359.

### Ability (Kemampuan)

| No | Item | Kode |
|---|---|---|
| 1 | [NAMA SISTEM/PENYEDIA] sangat kompeten dan efektif dalam menyediakan layanan | AB1 |
| 2 | [NAMA SISTEM/PENYEDIA] memiliki keahlian yang dibutuhkan untuk layanan yang dijanjikan | AB2 |
| 3 | Saya merasa [NAMA SISTEM/PENYEDIA] mampu menjaga keamanan transaksi/data saya | AB3 |

### Benevolence (Kebajikan)

| No | Item | Kode |
|---|---|---|
| 4 | [NAMA SISTEM/PENYEDIA] memiliki kepentingan yang baik terhadap saya | BE1 |
| 5 | Saya percaya [NAMA SISTEM/PENYEDIA] tidak akan menggunakan informasi saya untuk tujuan yang merugikan saya | BE2 |
| 6 | [NAMA SISTEM/PENYEDIA] peduli dengan kesejahteraan penggunanya | BE3 |

### Integrity (Integritas)

| No | Item | Kode |
|---|---|---|
| 7 | [NAMA SISTEM/PENYEDIA] jujur dalam berinteraksi dengan penggunanya | IN1 |
| 8 | [NAMA SISTEM/PENYEDIA] mematuhi peraturan dan standar yang berlaku | IN2 |
| 9 | Saya percaya [NAMA SISTEM/PENYEDIA] memiliki prinsip-prinsip yang dapat diterima | IN3 |

## 32.5 Template ECM — Kontinuansi (Bhattacherjee, 2001)

**Sumber:** Bhattacherjee, A. (2001). *MIS Quarterly*, 25(3), 351–370.

### Confirmation (Konfirmasi)

| No | Item | Kode |
|---|---|---|
| 1 | Pengalaman saya menggunakan [NAMA SISTEM] melebihi ekspektasi saya sebelumnya | CON1 |
| 2 | Layanan yang diberikan [NAMA SISTEM] lebih baik dari yang saya harapkan | CON2 |
| 3 | Secara keseluruhan, ekspektasi awal saya tentang [NAMA SISTEM] terkonfirmasi | CON3 |

### Post-Adoption Perceived Usefulness

| No | Item | Kode |
|---|---|---|
| 4 | Menggunakan [NAMA SISTEM] meningkatkan produktivitas saya (berdasarkan pengalaman nyata) | PU_POST1 |
| 5 | Menggunakan [NAMA SISTEM] meningkatkan efektivitas saya | PU_POST2 |
| 6 | Secara keseluruhan, saya merasa [NAMA SISTEM] berguna | PU_POST3 |

### Satisfaction (Kepuasan)

| No | Item | Kode |
|---|---|---|
| 7 | Saya puas dengan pengalaman menggunakan [NAMA SISTEM] | SAT1 |
| 8 | Saya merasa senang menggunakan [NAMA SISTEM] | SAT2 |
| 9 | Pengalaman menggunakan [NAMA SISTEM] terasa menyenangkan | SAT3 |

### Continuance Intention

| No | Item | Kode |
|---|---|---|
| 10 | Saya berencana untuk terus menggunakan [NAMA SISTEM] di masa mendatang | CI1 |
| 11 | Saya berniat untuk terus menggunakan [NAMA SISTEM] secara rutin | CI2 |
| 12 | Saya akan merekomendasikan orang lain untuk menggunakan [NAMA SISTEM] | CI3 |

## 32.6 Template Demografis (Standar)

### Data Demografis Responden

```
BAGIAN A: DATA RESPONDEN

A1. Usia:
    ☐ < 20 tahun
    ☐ 20–29 tahun
    ☐ 30–39 tahun
    ☐ 40–49 tahun
    ☐ 50–59 tahun
    ☐ ≥ 60 tahun

A2. Jenis Kelamin:
    ☐ Laki-laki
    ☐ Perempuan
    ☐ Lainnya / Tidak ingin disebutkan

A3. Pendidikan Terakhir:
    ☐ SD / Sederajat
    ☐ SMP / Sederajat
    ☐ SMA / Sederajat
    ☐ D3
    ☐ S1 / D4
    ☐ S2
    ☐ S3

A4. Pekerjaan:
    ☐ Pelajar/Mahasiswa
    ☐ Karyawan Swasta
    ☐ ASN/PNS
    ☐ Wirausaha/UMKM
    ☐ Profesional (Dokter, Pengacara, dll.)
    ☐ Ibu Rumah Tangga
    ☐ Lainnya: ___________

A5. Penghasilan per Bulan:
    ☐ < Rp 2.000.000
    ☐ Rp 2.000.000 – Rp 5.000.000
    ☐ Rp 5.000.001 – Rp 10.000.000
    ☐ > Rp 10.000.000
    ☐ Tidak ingin disebutkan

A6. Frekuensi Penggunaan [NAMA SISTEM]:
    ☐ Pertama kali / Baru mencoba
    ☐ Beberapa kali (< 1 bulan)
    ☐ Rutin (1-6 bulan)
    ☐ Sangat rutin (> 6 bulan)
```

## 32.7 Contoh Kuesioner Lengkap: Adopsi Mobile Banking

### Struktur Kuesioner (Mobile Banking versi TAM + Trust + BI)

```
KUESIONER PENELITIAN
"Faktor-faktor yang Mempengaruhi Adopsi Mobile Banking di Indonesia"

[Informed Consent — lihat template di BAB-31]

BAGIAN A: DATA RESPONDEN
[Gunakan template demografis di atas]

BAGIAN B: PENGALAMAN DENGAN MOBILE BANKING
B1. Aplikasi mobile banking yang Anda gunakan: ___________
B2. Sudah berapa lama menggunakan: ___________

BAGIAN C: INSTRUMEN PENELITIAN
Berilah tanda (✓) pada kolom yang sesuai dengan pendapat Anda.
1 = Sangat Tidak Setuju  2 = Tidak Setuju  3 = Netral  4 = Setuju  5 = Sangat Setuju

[PU1-PU5] — 5 item Perceived Usefulness (sesuaikan dengan "mobile banking")
[PEOU1-PEOU6] — 6 item Perceived Ease of Use
[AB1-AB3] — 3 item Trust/Ability
[BE1-BE3] — 3 item Trust/Benevolence
[IN1-IN3] — 3 item Trust/Integrity
[BI1-BI3] — 3 item Behavioral Intention

BAGIAN D: KOMENTAR TERBUKA (Opsional)
"Apakah ada hal lain yang ingin Anda sampaikan tentang pengalaman Anda menggunakan mobile banking?"
___________________________________________

TERIMA KASIH ATAS PARTISIPASI ANDA!
```

## ✅ Latihan Praktis

1. **Adaptasi:** Pilih satu template kuesioner di atas (TAM, UTAUT, atau UTAUT2) dan adaptasikan untuk meneliti adopsi **ShopeePay** oleh mahasiswa! Ganti item yang perlu diganti, tambahkan item khusus jika diperlukan!

2. **Evaluasi:** Review kuesioner TAM yang pernah Anda lihat dalam jurnal Indonesia. Apakah item kuesionernya sudah mengikuti item asli Davis (1989)? Apakah ada item yang tidak sesuai?

3. **Desain lengkap:** Rancang kuesioner **lengkap** (informed consent + demografis + semua konstruk) untuk penelitian adopsi **e-government** oleh ASN di instansi pemerintah! Gunakan teori yang paling tepat!
